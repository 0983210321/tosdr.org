API

We encourage people to develop applications and other services integrating our ratings in their work. This page should help you use our data for your own app.

For more information, please have a look at the README.

  • Each service has a JSON file, see http://tos-dr.info/services/
  • Each point has a JSON file, see http://tos-dr.info/points/
  • We also have an Index, see http://tos-dr.info/index/

All JSON files are published under the CC BY SA 3.0 license.
